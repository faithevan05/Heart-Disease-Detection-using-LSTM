# Heart-Disease-Detection-using-LSTM

##  Overview

This project uses a Long Short-Term Memory (LSTM) deep learning model to predict the presence of heart disease based on patient data.

##  Problem

Early detection of heart disease is critical but often requires expert analysis. Traditional methods may not efficiently capture complex patterns in medical data.

##  Solution

Developed a deep learning-based predictive model using LSTM to analyze patient health parameters and classify the likelihood of heart disease.

##  Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Pandas
* Matplotlib

##  Model Details

* Model: LSTM Neural Network
* Task: Binary Classification (Heart Disease / No Disease)
* Data preprocessing included normalization and feature selection

##  Results

* Achieved strong prediction performance on test data
* Demonstrates potential for clinical decision support systems

##  How to Run

1. Install required libraries:
   pip install numpy pandas matplotlib tensorflow

2. Run the notebook or script:
   python main.py
